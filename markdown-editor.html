使用帮助

这是一个即开即用的浏览器 Markdown 编辑器：左侧写作，右侧实时预览。内容会自动保存在浏览器本地，下次打开自动恢复。

开始写作

  • 在左侧输入 Markdown，右侧同步渲染。
  • 用工具栏按钮快速插入标题、加粗、列表、引用、代码、链接、图片、表格等。
  • 选中文字再点格式按钮，可自动包裹或替换选区。

调整视图

  • 拖动中间分隔线调整左右宽度；点击标题栏的折叠按钮收起或展开区域。
  • 工具栏「视图」可切换“编辑 + 预览 / 仅编辑 / 仅预览”。
  • 右侧顶部的「源码」标签可直接改源码，切回「预览」看效果。

保存、导入与导出

  • 按 Ctrl+S 或点「保存」按钮；编辑器每 500ms 自动保存一次。
  • 点「导入」导入本地 .md / .txt，或直接把文件拖进窗口。
  • 点「导出」导出 Markdown、HTML、Word、PDF（打印另存为）或图片。

图片与公式

  • 工具栏「图片」支持链接插入或本地上传；本地图片会转为 Base64 嵌入文档，方便离线使用。
  • 支持 $...$ 行内公式和 $$...$$ 块级公式。

网页转 Markdown

  • 点「网页转 MD」输入网页链接，可自动提取正文。
  • 知乎、微信公众号等建议勾选「使用本地代理」，先运行同目录的 web-to-md-proxy.py。
  • 需要登录的页面，可运行 python web-to-md-proxy.py --bookmark 生成书签小工具提取。

常用快捷键：Ctrl+S 保存，Ctrl+Z 撤销，Ctrl+Y / Ctrl+Shift+Z 重做，Ctrl+B 加粗，Ctrl+I 斜体，Ctrl+U 下划线，Ctrl+K 链接，Ctrl+Shift+K 图片，Ctrl+F 查找，Ctrl+H 替换，Tab 插入 4 空格缩进。
